# Daily Task Planner (Django)

A web-based task management system built using Django to help users organize and track daily tasks efficiently.

## 🚀 Features
- Add, update, and delete tasks
- Mark tasks as completed
- Dynamic task dashboard

## 🛠 Tech Stack
- Python (Django)
- SQLite
- HTML, CSS

## ⚙️ How It Works
- Django handles routing and backend logic
- Tasks are stored in a database
- User actions trigger CRUD operations via views

## 📊 Architecture
- Model: Task database schema
- View: Handles user requests
- Template: Displays UI
